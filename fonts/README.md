# Fonts for Lofi Video Thumbnails

This directory contains custom fonts for thumbnail generation.

## Required Fonts

1. **Instrument Serif** - For main title text

   - Download from: https://fonts.google.com/specimen/Instrument+Serif
   - File needed: `InstrumentSerif-Regular.ttf`

2. **Plus Jakarta Sans** - For subtitle text
   - Download from: https://fonts.google.com/specimen/Plus+Jakarta+Sans
   - File needed: `PlusJakartaSans-ExtraBold.ttf`

## Installation

1. Download the fonts from Google Fonts
2. Place the TTF files in this directory:
   - `fonts/InstrumentSerif-Regular.ttf`
   - `fonts/PlusJakartaSans-ExtraBold.ttf`

## Note

If font files are not found, the system will use fallbacks:

- Instrument Serif → Georgia (serif)
- Plus Jakarta Sans → Arial Black (sans-serif)
